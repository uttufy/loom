# loom status

Show context usage and stats

## Overview

Display current orchestrator status including:
- Context window usage
- Compaction statistics
- Active locks
- Session progress

## Usage

```bash
loom status
```

## Examples

```bash
# Show orchestrator status
loom status

# Output in JSON format
loom status --json
```
