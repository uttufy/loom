# loom config-docs

Generate config documentation

## Overview

Generate configuration documentation in Mintlify format.

The generated documentation includes:
- Config overview page
- Individual pages for each config section
- Option tables with types and defaults

## Usage

```bash
loom config-docs
```

## Examples

```bash
# Generate config documentation
loom config-docs
```
