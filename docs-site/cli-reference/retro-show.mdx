# loom retro show

Show retrospective details

## Usage

```bash
loom retro show [args]
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<retro-id>` | Yes | The retrospective identifier |

## Examples

```bash
# Show retrospective details
loom retro show retro-123
```
