# loom hooks test

Test hook execution

## Usage

```bash
loom hooks test [args]
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<event>` | Yes | The hook event to test |

## Examples

```bash
# Test a hook event
loom hooks test pre-tool-call
```
