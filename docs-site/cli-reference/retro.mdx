# loom retro

Create and view retrospectives

## Overview

Manage session retrospectives for learning.

Retrospectives capture:
- What tasks were completed/failed
- What strategies worked
- Lessons learned for future sessions

## Usage

```bash
loom retro
```

## Examples

```bash
# Create and view retrospectives
loom retro
```
