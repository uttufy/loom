# loom compact

Run importance-weighted compaction

## Overview

Run memory compaction to reduce context size.

Uses importance-weighted compaction that retains
more detail for high-dependency and failed tasks.

## Usage

```bash
loom compact
```

## Examples

```bash
# Run memory compaction
loom compact
```
