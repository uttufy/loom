# loom docs

Generate CLI documentation

## Overview

Generate CLI documentation in Mintlify format.

The generated documentation includes:
- Command overview page
- Individual pages for each command
- Usage examples and flags

## Usage

```bash
loom docs [flags]
```

## Flags

| Flag | Short | Default | Description |
|------|-------|---------|-------------|
| `--config` | `-c` | `` | config file (default is loom.yaml) |
| `--help` | `-h` | `false` | help for docs |
| `--json` | `-j` | `false` | output in JSON format |

## Examples

```bash
# Generate CLI documentation
loom docs
```
