# loom patterns

List learned patterns

## Overview

Display patterns learned from past sessions.

Patterns capture reusable strategies and approaches
that can be applied to similar tasks.

## Usage

```bash
loom patterns
```

## Examples

```bash
# List learned patterns
loom patterns

# Output in JSON format
loom patterns --json
```
