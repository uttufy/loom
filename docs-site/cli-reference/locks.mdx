# loom locks

Show current file locks

## Overview

Display all active file locks for multi-agent coordination.

File locks prevent conflicts when multiple agents
are working on the same codebase.

## Usage

```bash
loom locks
```

## Examples

```bash
# Show all active file locks
loom locks

# Output in JSON format
loom locks --json
```
